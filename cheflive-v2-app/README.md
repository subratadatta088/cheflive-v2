# cheflive-v2-app (Electron shell)

This folder contains the Electron "main process" that loads the UI from `../cheflive-v2-ui`.

## Development

```bash
cd cheflive-v2-app
npm install
npm run dev
```

## Production (local)

```bash
cd cheflive-v2-app
npm install
npm run build:ui
npm run start
```
